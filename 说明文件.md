# 安装node.js包

```
npm install -g gitbook-cli
```

```
gitbook init
```

```
npm init
```

```
gitbook serve
```

package.json

```
"scripts": {
    "serve": "gitbook serve",
    "build": "gitbook build"
  },
```

创建文件.bookignore

gitbook install
